Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language. Inside a host environment, JavaScript can be connected to the objects of its environment to provide programmatic control over them.

console.log('Hello World');
  • Client-side JavaScript
  • Server-side JavaScript
Variables

Variables are containers for storing data values.

let x = 5;

Three ways to declare variables:

var x = 5;\nconst y = 10;\nlet z = 15;
  • var - function scope
  • let - block scope
  • const - block scope, immutable
Data Types

JavaScript has 8 basic data types:

let message = "hello"; // string
  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • BigInt
  • Symbol
  • Object
Functions

A function is a block of code designed to perform a particular task.

function square(number) {\n return number * number;\n}

Arrow functions provide concise syntax:

const square = (number) => number * number;
Loops

Loops can execute a block of code multiple times.

for (let i = 0; i < 5; i++) {\n console.log(i);\n}

Different types of loops:

  • for
  • while
  • do...while
